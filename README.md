# SpringBoot-Project-WishBook
스프링부트를 이용한 복합 도서 사이트

## 프로젝트 소개
책 구매, 경매, 중고 거래, 공동 구매, 책 소모임 모집 등이 가능한 복합 도서 사이트입니다.
<br>

## 개발 기간
* 23.03.29 ~ 23.06.18

### 멤버 구성
  - 팀장 : 이도하 - 중고 거래 , 메인 페이지
  - 팀원 : 이지호 - 회원 , 공동 구매
  - 팀원 : 장현지 - 책 구매 , 경매
  - 팀원 : 김효민 - 장바구니, 구매, 독서 모임

  - 기능 위주로 업무를 분담
      : 통합하는 과정에서 같이 구현한 부분도 많음
  - 주요 설계는 공통 진행


### 개발 환경
- 'java 17'
**IDE** : STS 4.4.18
**Framework** : SpringBoot 2. 3 .0
**Database*** : OracleDb
**ORM** : Mybatis
